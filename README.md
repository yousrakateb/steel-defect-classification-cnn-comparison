# Steel Surface Defect Classification Using CNN Architectures: A Comparative Study

This repository contains the implementation of a comparative study of four convolutional neural network architectures for steel surface defect classification using the NEU-CLS dataset.

## Models Evaluated
- DenseNet-121
- EfficientNet-B0
- MobileNetV2
- NASNet-Mobile

## Main Features
- Transfer learning with ImageNet-pretrained weights
- 5-fold stratified cross-validation
- Comparative performance analysis
- Grad-CAM explainability visualization

## Dataset
The experiments were conducted on the NEU-CLS steel surface defect dataset.

## Reproducibility
All training, evaluation, and explainability scripts used in the study are provided to support reproducibility and transparency.
